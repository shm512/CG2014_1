Программа обрабатывает и анализирует изображения с целью нахождения на них "клада":
некоего объекта, отличающегося от фона, на который указывает последовательность стрелок,
первая из которых - красная. Реализованы следующие этапы:
 - предобработка (медианный фильтр)
 - бинаризация изображения (отсечение пикселей по порогу яркости, определяющемуся по гистограмме)
 - выделение связных компонент обходом в ширину
 - вычисление геометрических признаков связных компонент
 - выбор стрелок и потенциальных кладов по значению геометрических признаков
 - нахождение среди стрелок красной и обход изображения по направлению, указываемую
   стрелками, начиная с красной - пока не найдём "клад" (первый объект на пути, 
   не являющийся стрелкой)
 - путь отображается на изображении, а найденный клад обводится рамкой

Каталог pic содержит тестовые изображения, каталог labelling - правильные ответы к ним.
